The easiest way to add Facebook to your site.

== Description ==

The Facebook Like Button plugin lets users share pages from your site back to their Facebook profile with one click. The button is automitically added to your post. Other options include a shortcode or template tag ( for those theme hackers ).


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `the-facebook-like-button` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Facebook Like Button options section under Options.
